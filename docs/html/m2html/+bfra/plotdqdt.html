PURPOSE

PLOTDQDT Plot the log-log q vs dq/dt point-cloud

SYNOPSIS

DESCRIPTION 

PLOTDQDT Plot the log-log q vs dq/dt point-cloud 
 
 Syntax
 
     [hFits,Picks,Fits] = bfra.plotdqdt(q,dqdt)
     [hFits,Picks,Fits] = bfra.plotdqdt(_,'fitmethod',fitmethod)
