getEventsData

PURPOSE

GETEVENTSDATA get data from Events for event number eventTag

SYNOPSIS

DESCRIPTION 

GETEVENTSDATA get data from Events for event number eventTag

  Data = getEventsData(Events,eventTag) returns the data in Events struct
